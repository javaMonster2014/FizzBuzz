To run:
---------

JAR file: 
make sure both the JAR file and the name_of_input_file.txt are in the current 
working directory, or specify their absolute path.

java -cp FizzBuzz.jar Main name_of_input_file.txt

or

java -cp /.../FizzBuzz.jar Main /.../name_of_input_file.txt



Java file:

javac Main.java
java Main name_of_input_file.txt

*you may also use absolute paths as well!
